# Cyber Dogg

Веб-приложение для развлекательного центра Cyber Dogg.

## Технологии

### Frontend
- **Vue 3** (Composition API, `<script setup>`)
- **Vite** — сборка и dev-сервер
- **Vue Router** — маршрутизация с защитой роутов
- **CSS3** — кастомные стили с CSS-переменными, адаптивный дизайн

### Backend
- **Fastify** — Node.js фреймворк
- **MySQL** — база данных
- **EmailJS** — отправка email без SMTP
- **reservations.json** — хранение бронирований

## Установка и запуск

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
cp .env.example .env
# Заполните .env необходимыми переменными
npm run dev
```

## Страницы приложения

### Пользовательские страницы
- **Главная** — информация о заведении, бронирование, акции
- **Меню** — напитки и кухня
- **Цены** — прайс-лист по залам с расписанием
- **Бронирование** — форма заявки с валидацией
- **Подарочные сертификаты** — выбор типа сертификата и опций
- **Покупка сертификата** — оформление заказа
- **Авторизация/Регистрация** — вход, регистрация, восстановление пароля
- **Личный кабинет** — профиль пользователя, баланс игрового времени

### Админ-панель
- Управление пользователями (добавление/удаление)
- Добавление игрового времени пользователям
- Таблица заказов сертификатов с промокодами
- Смена статуса сертификатов
- Активация сертификатов на номер телефона

## Игровые залы

1. **Common Room** — 20 ПК
2. **Battle Arena** — 10 ПК
3. **VIP Room** — 5 ПК
4. **PlayStation 5** — до 5/7 человек

### Расписание работы
- **День** — 08:00 — 15:00
- **Вечер** — 15:00 — 22:00
- **Ночь** — 22:00 — 08:00

## Система сертификатов

### Типы сертификатов
- Common Room
- Battle Arena
- VIP Room
- PlayStation

### Опции
- Время суток: день/вечер
- Продолжительность: 3/5 часов
- Пакетные предложения: Кибер сутки (24ч), Ночь (8ч)
- PlayStation: выбор количества игроков (5/7 человек)

### Особенности
- Генерация уникальных 10-значных промокодов
- Статусы: pending (ожидает), paid (оплачен), used (использован), cancelled (отменён)
- Администратор может активировать промокод на номер телефона пользователя
- При активации время автоматически добавляется на баланс пользователя

## Безопасность

- Защита маршрутов (только для авторизованных пользователей)
- Ролевая модель (admin/user)
- Валидация форм на фронтенде и бэкенде
- Cascade delete — при удалении пользователя удаляется и его игровое время
